Aider

Paul Gauthier

开源终端 AI 结对编程工具，直接编辑本地文件并自动 Git 提交

访问官网

价格方案

Free
免费/月
  • 完全开源免费
  • 支持所有主流 LLM
  • 无限使用（需自备 API Key）

免费版限制：工具本身免费，需自付 LLM API 费用

核心功能

多 LLM 支持

兼容 Claude、GPT-4o、DeepSeek、Gemini 及本地模型

Git 原生

每次代码修改自动创建描述性 Git 提交

全仓库理解

自动分析项目结构，理解文件间依赖关系

IDE 集成

支持在 VS Code、Neovim 等编辑器中通过 AI 注释触发

优缺点分析

优点

  • 完全开源免费，无订阅费用
  • 支持几乎所有 LLM（Claude、GPT、DeepSeek、本地模型）
  • Git 原生：每次修改自动创建原子提交
  • 终端运行，与现有工作流无缝集成

缺点

  • 需要自备 LLM API Key（有 API 费用）
  • 纯命令行界面，学习曲线较陡
  • 没有图形界面，不适合可视化调试

适合与不适合

适合

• 熟悉终端的资深开发者

• 想要灵活选择 LLM 的用户

• 注重隐私不想代码上传云端

不适合

• 不熟悉命令行的初学者

• 需要图形化 IDE 体验的用户
